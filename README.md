## 자료구조란?

- 효율적인 접근 및 수정을 가능케 하는 자료의 조직, 관리, 저장을 의미한다. <br>
  더 정확히 말해, 자료 구조는 데이터 값의 모임, 또 데이터 간의 관계, <br> 그리고 데이터에 적용할 수 있는 함수나 명령을 의미한다.
  - 출처 : [위키피디아](https://github.com/blackb0x0714/CodingTest/tree/master/Array)
